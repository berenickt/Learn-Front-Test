## 사용 기술 스택

| Types      | Techs                                                |
| ---------- | ---------------------------------------------------- |
| Front      | react, tanstack Query, MUI, React Hook Form, zustand |
| Server     | express.js                                           |
| Build tool | vite                                                 |
| Test       | cypress, Testing Library, Storybook, MSW, Chromatic  |

## 설치 및 실행

```bash
nvm use # node 19.9.0 버전을 사용
npm ci # 의존성 설치
npm i
npm run dev # 노드 서버와 vite 개발 서버를 동시에 실행
```
